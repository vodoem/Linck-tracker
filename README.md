# Link Tracker

<!-- этот файл можно и нужно менять -->

Проект сделан в рамках курса Академия Бэкенда.

Приложение для отслеживания обновлений контента по ссылкам.
При появлении новых событий отправляется уведомление в Telegram.

Проект написан на `Java 23` с использованием `Spring Boot 3`.

Проект состоит из 2-х приложений:
* Bot (Telegram-бот и web-версия с авторизацией и шаблонами)
* Scrapper

Web-версия бота доступна по адресу `http://localhost:8080/ui/login`. Она предоставляет:

* регистрацию пользователей в H2 базе данных;
* вход через форму Spring Security;
* страницу чата на Thymeleaf, повторяющую команды телеграм-бота.

Для работы требуется БД `PostgreSQL`. Присутствует опциональная зависимость на `Kafka`.

Web-интерфейс использует встроенную БД H2, поэтому дополнительных настроек не требуется.
